# Incident Runbook

## Severity levels
| Level | Description |
|-------|-------------|
| SEV-1 | Full outage or critical customer impact |
| SEV-2 | Partial outage or major feature degradation |
| SEV-3 | Minor degradation with workaround available |

## Triage checklist
- [ ] Check deployment history
- [ ] Review logs and recent config changes
- [ ] Verify health endpoints
- [ ] Compare AWS and GCP service behavior
- [ ] Decide rollback or failover
